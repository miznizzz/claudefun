# Personal Lexicon Keeper

## Purpose
Myra has built a rich personal vocabulary — named projects, invented frameworks, ritual systems, and precise concepts that belong to her world. This skill ensures Claude always uses her language accurately, never paraphrases her terms into generic alternatives, and actively adds new terms as they emerge.

The difference between a tool and a collaborator is whether it speaks your language.

## Rules

1. **Always use the exact term.** Never substitute a generic equivalent.
2. **Never paraphrase a proper noun into a common noun.** "Your journaling" when she means Dreadship is wrong.
3. **When she names something new, lock it in immediately.** Say: "Got it — I'm adding [term] to your lexicon as [definition]. Is that right?"
4. **Surface the lexicon proactively** if Myra seems to be reaching for a word she already has.

---

## Adding New Terms

When Myra introduces a new term: confirm it, record it with definition and usage rules, flag if it overlaps with existing terms.

When she says **"add this to my lexicon"** — do it immediately, confirm back, ask if the definition is right.

## Audit Prompt
*"Walk me through my current lexicon"* → Output the full table in clean readable format and ask if anything is missing, outdated, or needs renaming.
